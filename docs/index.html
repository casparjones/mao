mao v0.1.3

paru, but friendlier.

curl -sSL https://casparjones.github.io/mao/install.sh | sh
// the idea

Why?

paru is great, but its pacman-style flags are hard to discover and easy to forget. mao gives them names: mao install firefox, mao update, mao search wine. Unknown flags still pass through to paru unchanged — your muscle memory keeps working.

// reference

Commands

mao paru does
$ mao update paru -Syu update system + AUR
$ mao update-db paru -Syy refresh package database
$ mao install <pkg> paru -S install package(s)
$ mao remove <pkg> paru -Rns remove pkg + deps + configs
$ mao search <q> paru -Ss search repos and AUR
$ mao info <pkg> paru -Si package details
$ mao list paru -Qe explicitly installed packages
$ mao list-all paru -Q all installed packages
$ mao orphans paru -Qtdq list orphaned packages
$ mao autoremove paru -Rns … remove orphans (with preview)
$ mao outdated paru -Qu show available updates
$ mao clean paru -Sc clean package cache
$ mao owns <file> paru -Qo which package owns a file
$ mao uninstall — remove mao + mao-tray
$ mao -Syu paru -Syu any paru flag passes through 1:1
// optional

mao-tray

A system tray companion that watches for updates in the background. Installed optionally — the installer will ask.

🕐

Hourly checks

Polls paru -Qu every hour and shows a desktop notification when updates are ready.

🔇

Silent updates

Runs paru -Syu --noconfirm in the background. Everything is logged to ~/.local/share/mao/tray.log.

⚙️

Auto-update toggle

Switch between notify-only and fully automatic mode from the tray context menu.

⏻

Update + Shutdown

One click to update silently and power off — useful before stepping away from the machine.

Requires python-pyqt6. Start manually with mao-tray &, or let the installer add an XDG autostart entry.

// getting started

Install

One-liner for Arch-based systems:

curl -sSL https://casparjones.github.io/mao/install.sh | sh

The installer will:

  • check you're on an Arch-based system
  • offer to install paru if it's missing
  • place mao in ~/.local/bin (or /usr/local/bin as root)
  • optionally install shell completions (fish / bash / zsh)
  • optionally set up mao-tray with autostart and sudoers

Prefer reading first? view install.sh

Manual install

curl -fsSLo ~/.local/bin/mao https://raw.githubusercontent.com/casparjones/mao/main/mao
chmod +x ~/.local/bin/mao

Uninstall

mao uninstall
# or without mao:
curl -sSL https://casparjones.github.io/mao/uninstall.sh | sh
// etymology

Why "mao"?

Mao (猫) is the sound a cat makes in Chinese — a nod to Maomao from Kusuriya no Hitorigoto. Like her: small, precise, gets the job done quietly.
